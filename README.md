# Social Media Sentiment Analyzer

This project extracts Reddit comments and performs sentiment analysis using OpenAI LLM.

## Features
- Fetch comments from Reddit
- Clean text data
- Sentiment classification (Positive / Negative / Neutral)
- Streamlit UI
- CSV download

## Setup

1. Clone repo:
   git clone https://github.com/yourusername/sentiment-streamlit-app

2. Install dependencies:
   pip install -r requirements.txt

3. Add secrets:
   Create `.streamlit/secrets.toml`

4. Run:
   streamlit run app.py

## Deployment
Deploy using Streamlit Cloud
